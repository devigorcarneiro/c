# 📑 Curso de Linguagem C — Professor Wagner Gaspar

> Repositório dedicado ao estudo de algoritmos, lógica de programação e estruturas de dados através da linguagem C.

Este espaço foi criado para armazenar todos os códigos, desafios e exercícios práticos desenvolvidos durante o curso completo de Linguagem C na Udemy, ministrado pelo professor Wagner Gaspar. O objetivo principal deste repositório é consolidar os fundamentos da computação de baixo nível, gerenciamento de memória e lógica estruturada, construindo uma base sólida para o desenvolvimento de software escalável e de alta performance.

## 🧠 Mentalidade de Desenvolvimento
* **Aprendizado Profundo:** Foco total na compreensão dos conceitos estruturais (como ponteiros, alocação dinâmica e manipulação de arquivos) sem atalhos.
* **Organização e Clareza:** Código limpo, bem documentado e estruturado de forma inteligível desde os primeiros scripts.
